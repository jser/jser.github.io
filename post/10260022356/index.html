O’Reilly Mediaで全てのebookとVideoが半額で、28日まで

2011年09月16日 Edit on GitHub 編集履歴を見る
  • Tweet
  • Pocket

O’Reilly MediaがBack to School Special - *Free to Choose* - Save 50% on All Ebooks & Videos - O’Reilly Mediaというキャンペーンで、全ての電子書籍と動画を半額で購入できるディスカウントコードを発行しています。

購入時に B2SDEALとディスカウントコードを入力する事で、50%OFFにて購入することができます。

また、 Programming | JavaScript | iPhone | Android | Python | HTML5 & CSS | jQuery | Java | PHP | Perl | Linux | Data といった人気のあるカテゴリーにおいて、下のような入門 | 中級 | 上級 に分別して書籍を紹介しているようです。

ディスカウントコードはSeptember 28thまでの有効期限となっているようです。
購入方法はO’Reilly(Japanも)が全ての電子書籍を半額で&売上を日本赤十字社に寄付 | JSer.infoなどを参考にどうぞ。
JavaScriptに限らず全ての電子書籍などが対象になっています。
