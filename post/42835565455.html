JSer.info #109 - jQuery 1.9.1 リリースされました。
主にbugfixについてで、1.9.x系への移行はjQuery Migrate pluginのWarning Messagesとupgrade guideに丁寧に色々書かれています。
Warning MessagesについてはjQuery Migrate plugin の使い方と警告文の自分なりのまとめ | 私的なjQueryも参考にするといいかもしれません。

Takazudolog - 地獄のvideo／audio要素という記事にてvideoとaudio要素について、各ブラウザの動作の違いから出る問題などについて書かれています。

ChromeのDev Toolsの機能について Chrome Dev Tools: JavaScript and Performance | Nettuts+や DevTools: Visually Re-engineering CSS For Faster Paint Time 等の記事で解説されています。
Chrome DevTools.next(紛らわしいけどDevtools.nextも)など最近Chrome Dev Toolsについての解説記事を見かけるようになってきた気がします。

また、Timeline Panelではrepaintを計測出来ますが、これをちゃんと役立てるには、reflow(layout)とrepaintそのものについても理解しておいたほうがよさそうな気がします。
以下のリソースが参考になると思います。

  1. How Browsers Work: Behind the scenes of modern web browsers - HTML5 Rocks ( Ja ) 必読書
  2. Getting Started With the WebKit Layout Code | Web Platform Team Blog (上の記事より少し内部より)
  3. Shiki’s Weblog: ESウェブブラウザ通信 - インクリメンタル リフロー #1 と #2(実際にCSSレンダリング エンジンを実装している段階で、リフローやリペイントなどについて段階ごとに解説されてる)
  4. ReFlowの原因とマークアップの最適化 « NAVER Engineers’ Blog (プラクティスよりだけど流れをしっかり組んでる)
  5. "遅いブログパーツを高速表示する方法"を検証する | ゆっくりと… (ブラウザ動作の理解と一緒に、計測する方法などについて)
  6. Fastersite: How (not) to trigger a layout in WebKit (reflow,repaintを起こすトリガーとなるメソッドのまとめ)
  7. Performance
