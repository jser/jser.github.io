JSer.info #111 - jQuery Mobile 1.3.0リリースされました。

JavaScriptのTest RunnerであるTestacular 0.6.0がリリースされました。
公式ページの方も色々変わっていて、Plusの部分にはTravis CIとの連携など他のTest Runnerでも参考になることが色々書かれています。
(Git Commit MsgにGitのコミット書式が書かれてるのも面白い)

Node.jsで動くタスク実行ツールのGrunt 0.4がリリースされました。
Plugins - Grunt: The JavaScript Task Runnerにプラグインがまとめられたり、ドキュメントページなども充実してきています。
実際にどのような変更があったかは Takazudolog - Grunt v0.4.0 での変更点 が参考になると思われます。
