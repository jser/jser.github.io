Apress 全ての書籍が$15、O'Reilly 一部書籍が60%オフ

2011年11月28日 Edit on GitHub 編集履歴を見る
  • Tweet
  • Pocket

O’Reilly MediaとApressがそれぞれ本日限定のセールをやっています。

Apress via kwout

Cyber Monday Deals of the Day - O’Reilly Media via kwout

Apressの方は全ての書籍が$15になるセールを
oreilly.comは新著やベストセラーな書籍を対象に60%オフになるCYBERMONDAYというディスカウントコードを発行しています。

O’Reilly Mediaはまあいつものことですが、Apressは結構Webやモバイル関係の書籍を結構出していたので、チェックしてみるといいかもしれません。

両方共、29日までなのでお早めに。
