2012-10-29のJS: JavaScriptパーサ Esprima 1.0.0、Orion 1.0、大規模CoffeeScript

2012年10月29日 Edit on GitHub 編集履歴を見る
  • Tweet
  • Pocket

JSer.info #94 - 既にいろいろなツールで使われていますが、JavaScriptパーサのesprima 1.0.0がリリースされました。

ちょうど、esprimaの作者でもあるAriya HidayatさんがBringing JavaScript Code Analysis to The Next Levelというesprimaなどに関するスライドを公開しているので、一緒に読むとesprimaによってどういうことができるのかやそれらに関連したツールはどういうものがあるのかが分かると思います。

ACEなどとよく並べられますが、ブラウザベースのJavaScript開発環境のOrion 1.0がリリースされました。こちらのツールも先ほどのesprimaが使われていたり、esprimaの中心的なContributorsでもあるConstellationさんが書いたJSDoc parserのdoctrineなどが使われています。

Aimingの大規模 JavaScript(CoffeeScript)開発についてのスライドや記事が公開されていて、色々な視点で書かれているので興味深いと思います。
