JSer.info #114 - Node v0.10.0のメジャーアップデートが公開されました。
なんで0.10.xなのといった疑問やどのような変更があったのかなどはNode.js-v0.10リリースアナウンスの簡単な解説と感想など - ぼちぼち日記がよくまとまっています。

Refactoring setInterval-based polling - Adventures in JavaScript Developmentという記事では、setIntervalで単純にXHRを叩いてPollingを行うコードの問題点を、ちゃんと通信の成否を確認してsetTimeoutでPollingを行うようにリファクタリングする内容が紹介されています。発展としてpromiseを使った方法も書かれていて、流れがあっていい感じの記事だと思います。

CSS Front-end Frameworks with comparison - By usabli.caというサイトでは、Twitter bootstrapや960 Grid SystemなどのCSSフレームワークの情報が簡潔にまとまっています。
