2012-11-06のJS:Yeti 0.2.13、Knockout 2.2.0、jQuery deprecated API

JSer.info #95 - Yeti 0.2.13がリリースされました。
QUnitのサポートがよくなりましたが、今後もテストフレームワークと連携していくために色々やっていくみたいです。
テストの並列実行とか色々細かい機能を取り入れてたりして面白いので期待してます。

jQueryの未来に備える | Cybozu Inside Out | サイボウズエンジニアのブログでjQuery 1.9あたりから削除され始めていく、deprecatedなAPIについてまとめられています。
liveやsizeなど意外とコードに残ってそうなAPIもdeprecatedになってるので見ておくといいかもしれません。
