JSer.info #60 - モバイルアプリケーション向けのフレームワークのSencha Touch 2.0がリリースされました。

Windows UpdateによるIEの自動アップグレードについて、日本でも3 月中旬よりやっていくとのことなので、IEのバージョン毎のシェアがまた少し異なってくるかもしれません。それにより今までWebではクロスブラウザの観点から使いにくかった機能が使える機会が増えるかもしれないので、それらを把握しておくといいです。

  • IE7 が死んだらできるいくつかのこと（ caniuse.com から見繕っただけ） - oogattaの勉強日記
  • IE 6, IE 7, IE 8 が退場した未来 - latest log

前から、地味にJavaScript IDEとして使える感じだったVisual StudioでのJavaScriptサポートが色々よくなったようなので、試してみるのもいいかもしれません。
