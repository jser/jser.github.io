2012-04-23のJS: JavaScript Enlightenment、WebStorm 4.0、mag.js

2012年04月23日 Edit on GitHub 編集履歴を見る

JSer.info #66 - “A book to turn a JavaScript library user into a JavaScript developer” という内容のJavaScript Enlightenmentが無料で公開されています。
以前は有料で販売されていたものだけあって、ちゃんと書かれていてサンプルコードそれぞれにjsfiddleで動くリンクが載っていて電子書籍って感じでよくできています。

JavaScript IDEのWebStorm 4.0が公開されました。
今回はあんまり派手な変更ではないですが、WebStorm指南書とかでも紹介してたように細かくブラッシュアップされています。(結構使い心地は違うかもしれないです).

ドイツのサイトですが、mag.jsというJavaScript Magazineが公開されていました。
とてもいい感じになってて羨ましい。

内容も関数型JavaScriptと尖った感じのだったり、例外処理の話だったり、JavaScriptクイズだったり、最近のライブラリ紹介や著名なライブラリの紹介、JavaScriptを学ぶ方法やコミュニティについてだったり、結構バランスよくできてるなーと感じました。
