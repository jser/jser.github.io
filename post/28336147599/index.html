JSer.info #81 - JavaScriptのMVC系(サバクラ両方で動く JavaScript の大規模開発を行うためにで解説されてるように、実際にはMVCとは違う形のものも多いですが)が最近乱立していますが、TodoMVCの話を中心にJourney Through The JavaScript MVC Jungleという記事が公開されています。(gistで公開されてた下書き的なものは消えたみたいです)

また、jQueryで書かれたものをリファクタリングしていくStep by step from jQuery to Backboneという記事は読みやすく出来てるので、Backboneを使った話ではありますが読んで損は無いと思います。
また、同じ著者が他にもBackboneについていくつか記事writings / publishedを公開してるみたいなので興味がある方はこちらも見るといいと思います。

原著がでてから1年ちょっとですが、ついにサイ本JavaScript 第6版の発売日が2012年08月10日に決まったようです。
