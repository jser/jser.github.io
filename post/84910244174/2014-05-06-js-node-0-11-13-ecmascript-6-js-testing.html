2014-05-06のJS: Node 0.11.13、ECMAScript 6入門、JS Testing Recipes

2014年05月06日 Edit on GitHub 編集履歴を見る
  • Tweet
  • Pocket

JSer.info #173 - Node v0.11.13 (Unstable)がリリースされました。 Unstableですが、このバージョンで内蔵してるV8が3.24.xに上がったので、以下で紹介されているES6の機能が使えるようになっています。

  • Node-v0.12からデフォルトでES6の一部が使えるようになった(WeakMap解説編) - ぼちぼち日記
  • Node-v0.12の新機能について

ECMAScript 6入门 という電子書籍(サイト)では、ECMAScript6で追加された新しい構文や既存のNumberやStringに増えたメソッドなどについて書かれています。

中国語の書籍ですが、ES6全般について書かれていて、コード例も多いので一度目を通してみるといいです。 また、文章はCC BY-NC 4.0でruanyf/es6tutorialに公開されていて、サイトはMarkdownで書いてdittoで作られていてワークフローとしても面白いと思います。

Nicholas C. Zakasさんも今 Github+ LeanpubでUnderstanding… by Nicholas C. Zakas [Leanpub PDF/iPad/Kindle]という書籍を執筆中です。

jstestやjsclassの作者である@jcoglanさんが書いたJavaScript Testing RecipesというJavaScriptのテストについての書籍が購入できるようになっています。

ブラウザ以外にもNode.jsアプリについてのテストについて詳しく書かれている感じで、NodeのCLIアプリのテストについて等も書かれています。
