2012-12-03のJS: enchant.js v0.6.0、Graphical Timeline、Async JavaScript

2012年12月03日 Edit on GitHub 編集履歴を見る
  • Tweet
  • Pocket

JSer.info #99 - enchant.js v0.6.0がリリースされ、デフォルトがCanvasベースの描画になったそうです。

FirefoxのアドオンとしてGraphical Timelineというネットワークのイベント、DOMイベント、GC等をタイムライン表示するものが公開されています(リリースされたのは7月で紹介するの忘れてた)
Chrome Developer Tools: Timeline Panelと類似ジャンルのもので、デザインや仕組みや使い方についてはTimeline Docs | An Idea ..にまとまっています。

Async JavaScriptという非同期JavaScriptをメインに扱った書籍が出るようです。(まだpre段階ですが購入はできます)
この書籍は以前LeanpubでAsync JavaScript - Recipes for Event-Driven Codeという題名で自費出版として販売していたものを改定したものとなるそうです。
著者のTrevor Burnhamは同じくPragmatic BookshelfからでてるCoffeeScript本の著者でもあります。

最近、DOM EnlightenmentやAddy Osmani氏がオライリーから出してる書籍等、一度違う所で電子書籍として公開や販売してから、改めて出版されるケースが増えてきている気がします。
特定のトピックが中心の書籍はこのパターンが多い気がするので、面白い流れなのかもしれないです。
