  • State of JavaScript 2022

言語機能やフレームワークなどについてのアンケートをとっていて、2022年12月15日まで受付けています。

過去の結果については次のページで公開されています。

  • The State of JS 2021
  • State of JS 2020

ECMAScript proposal: RegExp flag /v makes character classes and character class escapes more powerfulという記事では、現在ECMAScript Proposal Stage 3のRegExp v flagについて解説されています。

正規表現のvフラグは、正規表現に集合操作と構文を追加するProposalになっています。
記事では、vフラグを利用した際の文字クラスとUnicode Property Escape、文字クラスの集合操作、case-insensitiveマッチの改善について解説されています。
