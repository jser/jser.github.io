2023-09-01: Nuxt 3.7, Rome → Biome, Astro 3.0

2023年09月01日 Translator: rewrite0w0 Edit on GitHub See Revisions
  • Tweet
  • Pocket

JSer.info #658 - Nuxt 3.7 출시했습니다.

  • Nuxt 3.7 · Nuxt

Web Stream과 Response 지원, capo.js 사용한 <head> 내 요소 최적화 지원, Vite 4.4으로 업데이트, Nitro 2.6로 업데이트가 있습니다.

  • Announcing Nitro 2.6 · UnJS

tsconfig.json의 baseUrl가 아닌 상대 경로 사용하도록 변경, AsyncLocalStorage 사용한 비동기 컨텍스트 지원도 포함됩니다.

Rome는 Fork/아카이브화되고 Biome 재명명했습니다.

  • Announcing Biome

Rome Tools Inc. Rome 회사가 해산되고 연락두절이 되어, 과거 코어 팀이 Biome로 새롭게 프로젝트를 살린다 발표했습니다.

Astro 3.0.0 출시했습니다.

  • Astro 3.0 | Astro
  • Release astro@3.0.0 · withastro/astro

Node.js 16 지원 종료, ~/assets import alias가 기본적 포함되지 않도록 변경, class:list 동작 변경, 이미지 최적화에 Sharp 사용하는 변경이 있습니다.
또한, Scoped Style에 :where가 아니라 data 속성 사용하도록 변경, Experimental이던 View Transition와 Image 기능

  • View Transitions 🚀 Astro Documentation

옵션 삭제하고 기본값으로 유효상태로 변경했습니다.
그 외로는, React Fast Refresh가 작동되도록 수정했습니다.

Astro v2에서 마이그레이션 가이드도 공개중입니다.

  • Upgrade to Astro v3 🚀 Astro Documentation
