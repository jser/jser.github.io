2023-08-10: Lighthouse v11.0.0, Astro 2.10(View Transitions), Sharp의 WebAssembly 빌드

2023年08月10日 Translator: rewrite0w0 Edit on GitHub See Revisions
  • Tweet
  • Pocket

JSer.info #655 - Lighthouse v11.0.0가 출시되었습니다.

  • Release v11.0.0 · GoogleChrome/lighthouse

Node.js 16 지원 종료, 기본적으로 사용되지 않은 resource-summary/service-worker/first-contentful-paint-3g의 Audit이 삭제되었습니다.
또한, INP의 Audit가 Stable이 되었습니다.

Astro 2.10가 출시되었습니다.

  • Astro 2.10: Persistent State in View Transitions | Astro

Astro 2.9에 실험적 지원한 View Transitions에 추가하는 형태로, 지정한 요소를 영속적으로 만드는 transition:persist 명령어가 추가되었습니다.

또한, Astro 3.0가 베타 출시되었습니다.

  • astro/packages/astro/CHANGELOG.md at refs/heads/next · withastro/astro

Bringing Sharp to WebAssembly and WebContainers에서는, Native Addon인 이미지 처리 라이브러리 sharp를 WebAssembly으로 어떻게 빌드했는가 다룹니다.

emnapi를 사용해 Node API 이식, libvips를 Wasm으로 컴파일, 기동할 때 처리를 동기적으로 하기 위해 Emscripten를 어떻게 수정했는가에 대했는지도 다뤄집니다.

다음 Pull Request에 sharp 변경을 확인가능합니다.

  • WebAssembly build by RReverser · Pull Request #3522 · lovell/sharp
