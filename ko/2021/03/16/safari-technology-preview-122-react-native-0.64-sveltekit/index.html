JSer.info #531 - Safari Technology Preview 122 릴리스 됐다.

  • Release Notes for Safari Technology Preview 122 | WebKit

Safari Technology Preview 122에서 여러가지 기능이 구현됐다.
실험 플래그가 붙어 CSS :focus-visible와 CSS Color 5의 color-mix() / color-contrast() / Relative color syntax가 구현됐다.
ES Proposal Stage 3의 Private method, Static private methods, Top-Level await, RegExp Match Indices를 지원한다.
Web Worker와 Service Worker에서 type: module를 지원하고 Paint Timing의 (재)유효화 등도 포함됐다.

  • Threading the web with module workers
  • Chrome 80부터 Web Worker (Dedicated Worker)에서 ES Modules를 쓸수 있어요(일본어)

Safari와 직접적인 관계는 없지만 2021년 6월 즈음 릴리스 예정인 ECMAScript 2021의 후보도 공개됐다.

  • Release ES2021 Candidate March 2021 · tc39/ecma262
  • ECMAScript 2021: the final feature set

SvelteKit의 소스 코드가 공개됐다.
SvelteKit은 Svelte 기반의 Next.js와 같은 프레임워크였던 Sapper를 전환할 목적으로 만들어졌다.

  • sveltejs/kit: A monorepo for SvelteKit and friends

아직 소스 코드를 공개했을 뿐이며 아직 릴리스는 하지 않은 상태다.
What is SvelteKit? | SK Incognito라는 튜토리얼 사이트에 SvelteKit이 어떠한 프레임워크인지 소개 돼 있으므로 흥미가 있는 사람은 함께 살펴보길 바란다.

React Native 0.64가 릴리스 됐다.

  • Announcing React Native 0.64 with Hermes on iOS · React Native

AOT를 활용한 사전 컴파일로 footprint를 최적화 하여 애플리케이션의 시작 속도 등을 개선하는데 목적을 둔 Hermes JS Engine가 iOS에서도 사용할 수 있도록 됐다.
또, Hermes에 ES2015의 Proxy object 지원이 추가됐다.
그 외로 Metro에서 Inline Requires가 기본적으로 활성화, React 17로의 업데이트 등의 내용도 포함됐다.

Chromium Blog: Chrome 90 Beta: AV1 Encoder for WebRTC, New Origin Trials, and More

blog.chromium.org/2021/03/chrome-90-beta-av1-encoder-for-webrtc.html

Chrome ReleaseNote

Chrome 90 Beta 릴리스.
AV1 Encoder 지원.
Origin Trial로 getCurrentBrowsingContextMedia(), Breakout Box, WebAssembly Exception Handling을 지원한다.
CSS로는 aspect-ratio를 사용했을 때 width 또는 height 중 하나가 지정돼 있을 경우 크기(dimension)가 자동으로 추론되도록 개선됐고, :state 의사 클래스와 overflow: clip 속성을 지원한다.
Feature-Policy를 대신하는 Permissions-Policy, StaticRange 지원하며 navigator.plugins와 navigator.mimeTypes가 빈 값을 반환하도록 변경됐다.
