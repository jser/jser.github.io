JSer.info #581 - Redux Toolkit 1.8.0가 릴리스 됐다.

  • Release v1.8.0 · reduxjs/redux-toolkit

1.8.0에서 새로운 비동기 Middleware로 createListenerMiddleware가 추가됐다.

  • createListenerMiddleware | Redux Toolkit

createListenerMiddleware는 redux-saga나 redux-observable 등과 같이 비동기 처리를 다루기 위한 Middleware다.
단순한 작업을 다룰 수 있는 dispatch나 getState 등의 API 뿐 아니라 redux-saga에 영감받은 take, Temporal에 영감받은condition 등의 API도 제공해 복잡한 작업도 다룰 수 있도록 개선됐다.

아래 영상을 통해 이 createListenerMiddleware가 어떻게 만들어졌는지 소개하고 있다.

  • Redux Insights: the Redux Toolkit "action listener" middleware - YouTube
  • New experimental "action listener middleware" package available · Discussion #1648 · reduxjs/redux-toolkit

Node.js 17.6.0가 릴리스 됐다.

  • Node v17.6.0 (Current) | Node.js

Node v17.5.0에서 Streaｍ과 관련된 ECMAScript Proposal Stage 2 단계인 Iterator Helpers가 구현됐다.
하지만, 사양에 근거해 구현할 경우에 mongoose 등의 모듈이 동작하지 않게 되는 회귀가 발생하여 사양에 근거한 몇 가지 변경 내용을 되돌렸다(revert).

  • Regression in Node 17.5, Assigning a function to prototype of an Object results in a TypeError: Cannot assign to read only property 'x' of object 'y' at Object. · Issue #41926 · nodejs/node

구체적으로 Iterator Helper의 사양에서는 map 등의 메서드를 { [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: false }로 구현하도록 돼 있다.

이 경우에 다음과 같이 Stream의 prototype.map을 덮어쓰게 되면 Writable가 false이기 때문에 예외가 발생하므로 불가능하다.

// [Writable]]: false인 경우 덮어쓸 수 없다.
ReadableStream.prototype.map = function(fn) { // 예외가 발생한다.
  // ..
};

mongoose 등의 구현부에 이러한 대입이 존재하기 때문에 [[Writable]]: false를 [[Writable]]: true로 변경(revert)하는 내용이 Node.js 17.6.0에 포함됐다(예상과 다르게 BREAKING CHANGE 사항이 됐기 때문)

mongoose 측에서 이 문제에 대응한 내용은 다음과 같다.

  • fix Node.js 17.5 compatibility by benjamingr · Pull Request #11381 · Automattic/mongoose

A Complete Guide to CSS Cascade Layers | CSS-Tricks - CSS-Tricks에는 Chrome / Edge 99+, Firefox 97+, Safari Technology Preview 133에서 구현된 CSS Cascade Layers가 소개돼 있다.

CSS에는 UA, !important, 명시도 등에 의해 어느 스타일을 적용하는 게 맞는지에 대한 Cascading 사양이 정해져 있다.
이 사양에 Cascade Layers라는 내용이 추가돼 각 브라우저에서 구현이 진행되고 있다.

이 글에는 Cascade Layers의 기본적인 사용 방법, Important layers, Layer 우선도, sub layer 개념 등에 대해 작성돼 있다.
