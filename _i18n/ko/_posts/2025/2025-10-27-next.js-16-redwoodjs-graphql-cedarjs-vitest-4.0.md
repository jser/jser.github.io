JSer.info #753 - Next.js 16가 출시되었다.

- [Next.js 16 | Next.js](https://nextjs.org/blog/next-16)

Node.js 18지원 종료, AMP, `next lint`, `middleware.ts` 삭제 같이 파괴적인 변경이 포함됩니다.
새로운 기능으로, Cache Components에 의한 명시적 캐시제어와 PPR 통합, Next.js Devtools MCP 추가, `middleware.ts`를 `proxy.ts`으로 변경이 있습니다.
또한, Turbopack가 기본적 번들러가 되며, React Compiler 통합, 라우팅・프리페치 최적화, React 19.2 지원도 추가됩니다.

---

RedwoodJS 개발 종료, CedarJS으로 이행이 권장됩니다.

- [Release v8.9.0 · redwoodjs/graphql](https://github.com/redwoodjs/graphql/releases/tag/v8.9.0)
- [CedarJS | The React + GraphQL Web App Framework](https://cedarjs.com/)
- [cedarjs/cedar: The React + GraphQL Web App Framework](https://github.com/cedarjs/cedar)

CedarJS는 RedwoodJS를 fork한 React + GraphQL의 Web 애플리케이션 프레임워크로, 앞으로는 이 개발이 계승됩니다.

또한, Redwood 자체는 [RedwoodSDK](https://github.com/redwoodjs/sdk)라 불리는 Cloudflare Workers 대상 React 프레임워크로 집중됩니다.

---

Vitest 4.0가 출시되었다.

- [Vitest 4.0 is out! | Vitest](https://vitest.dev/blog/vitest-4)

Browser Mode가 stable이 되어, 비주얼 리그레션 테스트하는 `toMatchScreenshot` 지원, Playwright Trace와 frameLocator 추가가 포함됩니다. 또한, `expect.assert`와 `expect.schemaMatching`가 추가되었습니다. 파괴적인 변경으로, `basic` 레포트 삭제, Browser Provider 설정 변경, `workspace` 옵션 삭제, `coverage.all`와 `coverage.extensions` 삭제가 포함됩니다.
