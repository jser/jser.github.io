JSer.info #136 - Bootstrap 3 RC2(aka. Twitter Bootstrap) がリリースされました。
細かい部分のアップデートやMigrating from 2.x to 3.0等のドキュメントが整備されてきています。 

WIP: Bootstrap 3 by mdo · Pull Request #6342 · twbs/bootstrap にTodoとして残っている部分も大分減ってきています。

New Features of Firefox Developer Tools: Episode 25 ✩ Mozilla Hacks – the Web developer blog(訳: Firefox 開発ツールの新機能: エピソード 25 | Mozilla Developer Street (modest))では、
Firefox25での開発ツールの新機能についてまとめられています。 
ライブラリのブラックボックス化 という機能では、デバッガーなどが指定したファイル(ライブラリ)などを無視できるようになってます。
これを使うとステップ実行時にライブラリをスルーしたりできるので結構便利そうです。

  • Callable entities in ECMAScript 6
  • Toward Modern Web Apps with ECMAScript 6 | Blog | Sencha

どちらの記事も、今までのES5での書き方とES6の新しい文法を使った書き方を対比しながら、ES6について書かれています。
コンストラクタ関数とnew、thisやスコープといった混乱しやすい(対処方法も複数存在してる)部分をES6で導入された文法を使うことで、
目的が分かりやすくクリア(人によって左右されにくい書き方)になるかについて書かれています。
