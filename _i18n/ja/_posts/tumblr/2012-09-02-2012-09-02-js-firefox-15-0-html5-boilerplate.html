JSer.info #86 - Firefox 15.0がリリースされました。
キーイベント関係周りの実装が色々されていたり、パフォーマンス測定などをより高精度に取れるperformance.now()も実装されています。

  • Firefox 15 for developers | Mozilla Developer Network

HTML5 Boilerplate 4.0.0がリリースされました。
MITライセンスが追加されたこととFormatting and consistency changesやChange image replacement technique (#1149)や
ディレクトリやファイルのリネーム等、機能追加というよりは改善が中心のようです。

Javascript Scope QuizでJavaScriptのスコープについて、クイズ形式で学んでいけるようなサイトが公開されていました。
クイズの解答と一緒に解説も出るので結構分かりやすい気がします。
他のサイトですが、クロージャーのチュートリアル（What’s a Closure?）というクロージャーについてだけ扱う感じのものがあったり、
範囲を絞ってJavaScriptについて学べるコンテンツが出るのは面白い気がします。
